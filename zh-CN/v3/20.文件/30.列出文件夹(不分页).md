#### 列出文件夹下所有文件

```POST``` ```/v3/files/list/```

```json
{
	"path": "/16383",
	"start": 1,
	"limit": 4,
	"orderby": [["ctime", "asc"], ["directory", "desc"]]
}
```

或者

```POST``` ```/v3/files/list/```

```json
{
	"identity": "59e78e18e334cc643299334c45183df4",
	"start": 1,
	"limit": 4,
	"orderby": [["ctime", "asc"], ["directory", "desc"]]
}
```

或者使用 ```GET``` 方式

```GET``` ```/v3/files/list/59e78e18e334cc643299334c45183df4?start=4&limit=1&orderby=ctime```


#### 参数

* path 目录
* identity 目录ID
* start 跳过的条数
* limit 返回最大记录数
* orderby 排序
