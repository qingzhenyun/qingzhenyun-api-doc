# 列出回收站文件
件

> 列出当前回收站内容

#### 复制文件或文件夹使用的参数

|  参数  	|  类型  	| 可选 	|           说明           	|
|:------:	|:------:	|:----:	|:------------------------:	|
|  name  	| string 	|  是  	|  搜索的记录名字  	|
|  start  	| int64	|  是  	|        跳过记录数，从0开始    	|
| limit 	| int64 	|  是  	| 返回最大记录数，默认20 	|


```POST``` ```/v3/trashCan/list/```


```json
{
	"start": 1,
	"limit": 4,
	"orderby": [["ctime", "asc"], ["directory", "desc"]]
}
```

或者

```GET``` ```/v3/trashCan/list/?start=1&limit=4```
